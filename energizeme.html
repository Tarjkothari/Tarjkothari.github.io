About the Project

EnergizeMe is a comprehensive wellness platform developed to offer integrated solutions for personal health. It combines gym management with personal wellness tracking.

Key Features
  • Integrated Health Tracking: Tracks fitness progress, mental health metrics, and nutrition logs.
  • Habit Tracker: A backend system to help users monitor and maintain daily healthy habits.
  • Backend Architecture: Robust PHP backend to handle user data securely.
